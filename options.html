Connection Notes

Use this section as a quick reminder while setting everything up.

Notion token

This is secret and should stay private. It belongs in the hidden Integration token field.

Notion Database ID

Open the database as a full page and copy the 32-character ID from the URL.

Google Client ID

This is public client metadata, not a password. Paste the Client ID into the OAuth Client ID field.

Google Client Secret

Copy this from the OAuth client details in Google Cloud. Keep it private and paste it into the hidden Client Secret field.

Redirect URI

Add the redirect URI shown above into Google Cloud OAuth credentials before authorizing Google.
